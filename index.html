Music Theory Chord Voicings Exercise

Demo: https://andrewdyquia1.github.io/voicings-music-theory-exercise/

Github Repo

This is an exercise meant for student musicians and musicians who want to brush up on their ear training. It is in the form of a quiz with a canvas drawing out the grand staff, notes, and accidentals. The quiz plays a number of audio files representing music chords and displays the lowest pitch note. The user must then figure out the rest of the notes using that information.

Technologies Used:
  • HTML
  • SCSS
  • Javascript
